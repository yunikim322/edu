CHAPTER 1

응급상황 대응체계

응급상황 발생 → 즉시 상태 파악 → 119 신고 또는 관리자 보고 → 응급처치 → 이송 · 기록
위급한 경우
즉시 119 신고
구급차 도착까지 응급처치 시행 → 가능하면 직원 동승 이송 → 센터에 이송 병원 보고 → 센터에서 보호자 연락
↓
위급성이 낮은 경우
응급처치 후 관리자 보고
센터장·관리자 지시에 따라 병원 이송 여부 결정
1
절대 혼자 판단 금지
어떤 상황이든 반드시 관리자에게 즉시 보고하세요
2
반드시 기록
발생 내용과 후속 조치를 응급상황발생기록대장에 기록

CHAPTER 2

응급처치 기본 원칙

1
의료인에게 인계 전까지만
요양보호사는 의료 행위 불가. 의약품은 외용약·상비약 범위 내에서만 허용
2
침착하고 신속하게
가장 위급한 어르신을 우선으로 대응
3
함부로 이동 금지
골절·뇌졸중 등에서 잘못된 이동은 더 큰 손상을 유발
4
복약 정보 미리 메모
평소 복용 약·병원 정보를 메모해 응급 시 병원에 제출. 구토물·원인 약물도 함께 가져가기
5
모든 상황 기록 필수
응급상황 발생 내용과 처치 내용을 반드시 기록에 남기기

CHAPTER 3

즉각 대응 필수 ★

아래 상황은 즉각 대응이 생명을 결정합니다. 망설이지 마세요.
낙상 넘어짐·쓰러짐
확인 신호
어르신이 넘어지거나 쓰러짐 / 통증 호소 / 출혈
1
어르신 안정 — 의식 있으면 어떻게 넘어졌는지, 어디 아픈지 여쭤보기
2
통증 있으면 억지로 구부리거나 펴지 않기 — 골절 악화 위험
3
관리자 보고 + 가족·보호자 연락
4
출혈 있으면 지혈 → 환부 부목 고정 → 의료기관 이송
질식 기도폐쇄
확인 신호
갑자기 말 못 함 / 두 손으로 목을 감싸쥠 / 입술·얼굴이 파랗게 변함(청색증)
1
입 안에 보이는 이물질 즉시 제거
2
등 어깨뼈 사이를 손바닥으로 세게 5회 두드리기
3
하임리히법 (의식 있는 경우) — 어르신 뒤에 서서 허리 감싸안고, 배꼽과 명치 사이에 주먹 대고 위로 힘차게 밀어올리기. 이물질 나올 때까지 반복
4
의식 없는 경우 — 즉시 119 신고. 바닥에 눕히고 CPR(가슴압박 30회) 시작 → 입안 이물질 확인·제거. 이물질 나올 때까지 반복
5
청색증 심하거나 호흡 미회복 시 즉시 119 신고
경련 발작·경기
확인 신호
몸을 부들부들 떨거나 경직 / 의식 흐림
1
주변 위험한 물건 치우기 / 허리벨트·단추 풀기
2
머리 아래 부드러운 것 받치기 / 기도 확보
3
경련 중 억지로 붙잡거나 입에 물건 넣지 않기
4
대부분 1~2분 내 멈춤 — 경련 양상 관찰하며 대기
5
경련성 질환 없던 어르신 / 5분 이상 지속 시 즉시 119 신고 후 이송
심장·호흡 관련 응급
심장마비 흉통·맥박 없음
확인 신호
가슴을 짓누르는 심한 통증 / 청색증 / 식은땀·구역감 / 불규칙한 맥박 / 호흡곤란
1
활력징후 측정 / 반좌위(상체 약간 올린 자세)로 눕히기
2
목·가슴·허리 조이는 옷 풀기 / 금식 유지
3
니트로글리세린 응급약 있으면 혀 밑에 넣기
4
즉시 병원 이송
5
호흡·맥박 없으면 즉시 CPR 시작 ↓
심폐소생술 심폐소생술
호흡·맥박이 없으면 즉시 CPR을 시작하세요. 1분이 생명입니다.
1
어깨 두드려 반응 확인 → 없으면 즉시 119 신고 + AED 요청
2
딱딱한 바닥에 눕히기 / 압박 위치: 양쪽 젖꼭지 사이 흉부 정중앙
3
두 손 겹쳐 분당 100~120회, 깊이 약 5cm로 30회 압박
4
기도 열기(이마 누르기+턱 들기) → 코 막고 인공호흡 2회
5
가슴압박 30회 + 인공호흡 2회 반복 → 119 도착까지 멈추지 않기. 2인 구조 시 5주기마다 교대
AED 사용법
기기 켜기 → 패드 붙이기(오른쪽 쇄골 아래 / 왼쪽 젖꼭지 아래 옆구리) → 심장 리듬 분석 중 어르신 몸에 닿지 않기 → 충격 버튼 누르기 → 즉시 CPR 재시작

CHAPTER 4

뇌졸중·골절·외상·화상

골절 뼈 부러짐
확인 신호
통증 호소 / 팔다리 모양 이상 / 움직임 제한
1
함부로 움직이지 않기 — 즉시 119 연락
2
편평한 곳에 눕혀 불필요한 움직임 막기
3
출혈 있으면 지혈 → 거즈로 느슨하게 감싸기
4
부목으로 다친 위아래 관절 함께 고정
뼈가 피부 밖으로 나온 경우 절대 밀어 넣지 않기. 감염 위험 — 그대로 즉시 이송
외상 베임·찢어짐·관통상
확인 신호
날카로운 것에 베임 / 출혈
1
깨끗한 천·거즈로 눌러서 지혈
2
출혈 멈추면 약한 수돗물로 상처 세척 (입으로 빨지 않기)
3
지혈 후 부목 고정
절단 시
생리식염수 세척 → 거즈로 감싸 비닐 밀봉 → 얼음물 보관(직접 접촉 금지) → 즉시 이송
화상 열·불·뜨거운 것
확인 신호
뜨거운 것에 직접 닿음 / 피부 빨개짐·물집·극심한 통증 / 심한 경우 피부가 검게 타거나 하얗게 변함
1
찬물로 15~20분 식히기 — 얼음 직접 대지 않기, 세게 틀지 않기
2
옷은 가위로 잘라내기 (잡아당기지 않기) / 반지·시계·팔찌 즉시 제거
3
1도(빨개짐) — 찬물로 통증 완화
2도(물집) — 물집 터뜨리지 않기. 병원 소독·드레싱 필수
3도(검게 타거나 하얗게 변함) — 상처 건드리지 않고 즉시 이송
뇌졸중 골든타임 중요
확인 신호
한쪽 팔·다리 힘 빠짐 / 입가 처짐·말 어눌해짐 / 심한 두통·구토 / 한쪽 눈 안 보임
1
머리 움직이지 않게 하며 안전한 곳으로 이동
2
마비 없는 쪽이 아래로 가게 옆으로 눕히기 (토사물 흡인 방지)
3
머리·어깨 약간 올리고, 다리는 낮추기 (뇌압 상승 방지)
4
즉시 119 신고 — 골든 타임 이내 이송

CHAPTER 5

혈압·혈당·기타

호흡곤란 숨쉬기 힘들어함
1
반좌위·좌위 자세 (등 약간 기댄 자세) — 호흡 도움
2
곁에 머물며 손 잡고 안정, 심호흡 같이 하기
3
상태 지속 관찰 — 호흡이 점점 나빠지면 즉시 119 연락 준비
4
호흡곤란 지속 시 119 신고 후 이송
고혈압 두통·어지러움
기준
혈압이 최고혈압 160 / 최저혈압 90 이상, 또는 평상시보다 현저히 높음
1
혈압 측정 / 머리 약간 올린 자세로 안정
2
의식 없으면 음식·음료 드리지 않기
3
안정 후에도 혈압 안 내려가면 119 연락
기립성 저혈압 자세 바꿀 때 어지러움
확인 신호
자리에서 일어나다 어지러움 / 얼굴 창백·식은땀·휘청거림
1
머리 낮추고 다리 올린 자세로 눕혀 안정
2
저혈압 지속 시 즉시 119 연락
예방: 평소 자리에서 일어날 때 천천히, 단계적으로 자세 바꾸도록 도와드리기
저혈당 식은땀·어지러움 (당뇨 어르신)
기준
혈당 70mg/dL 미만 / 식은땀·허기·기력 없음·의식 흐림
1
의식 있고 삼킬 수 있으면 — 설탕물·오렌지주스·초콜릿·사탕 등 당분 섭취
2
보통 15분 이내 호전
의식 없거나 삼키지 못하면 음식 드리지 않기 — 질식 위험. 즉시 119 연락
염좌 발목·손목 삠
1
출혈 있으면 압박 지혈
2
다친 부위를 심장보다 높이 올리기
3
냉찜질 20~30분 — 붓기·통증 완화
4
탄력붕대·부목으로 고정 후 안정
일사병 여름철 더위 노출
확인 신호
더운 곳 노출 후 구역질·어지러움·두통
1
즉시 시원한 곳으로 이동 / 편안한 자세
2
젖은 수건으로 감싸고 부채질 / 얼음주머니를 겨드랑이·목·사타구니에 대어 체온 내리기
3
의식 있으면 이온음료·물 마시게 하기
4
의식 없으면 음식 드리지 않고 즉시 119 연락
